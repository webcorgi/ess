# ESS CARE 대시보드 시각화 설계안

### [1] 인트로: ESS CARE의 목적과 시각화의 필요성

현재 상황

- ESS CARE는 에너지 저장장치의 실시간 상태·이상 정보·운영 데이터를 관리하는 플랫폼
- 데이터 테이블 중심으로 운영자·관리자 입장에서 한눈에 상태 파악이 가능하도록 제작되어야함

이번 회의의 목적
 "정보 노출을 최소화하면서도 직관적인 시각화 대시보드 구조를 제안"

핵심 포인트: "데이터는 그대로지만, 표현 방식이 달라지면 관리 효율이 달라진다."

---

### [2] 대시보드 전체 콘셉트 제시

"운영자 중심의 실시간 모니터링 + 관리자 중심의 정보 노출 제어를 결합한 하이브리드 대시보드"

### [3] UI 구조 설명

"한 화면에서 ESS의 상태, 흐름, 이상 상황을 한눈에 확인할 수 있게 한다."

---

### [4] 시각화 전략 설명

"수치 중심 → 관계 중심 → 패턴 중심"으로 단계적 인식 구조를 설계

| 단계        | 목적                          | 표현 방식 예시          | 시간 |
| ----------- | ----------------------------- | ----------------------- | ---- |
| ① 즉시 인식 | 현재 ESS 상태(정상/이상/경보) | 카드 색상 / 아이콘 변화 | 1초  |
| ② 관계 인식 | 입출력 흐름, 전력 흐름        | 그래프, 비교 차트       | 3초  |
| ③ 패턴 인식 | 장기적인 트렌드, 이상 탐지    | 테이블                  | 5초  |

설계 원칙: "시각적 계층 구조"를 통해 정보를 단계적으로 노출함으로써, 한눈에 파악 + 클릭 시 상세 진입이 가능하도록 구성

---

### [5] 기술적 구현 가능성 설명

"이 시각화는 실제로 React + D3.js(or Recharts) + Tailwind로 구현 가능하며, Socket 기반 실시간 데이터 수신 구조를 갖춘다."

#### 기술 구조

| 영역           | 기술 스택                                     | 설명                                |
| -------------- | --------------------------------------------- | ----------------------------------- |
| Frontend       | Next.js (App Router) + Recharts + TailwindCSS | 반응형 대시보드 UI                  |
| Data Source    | WebSocket → JSON feed                         | 실시간 SOC, SOH, Power, Temp 데이터 |
| Access Control | Auth or Role-based Middleware                 | 권한별 정보 노출 제어               |
| Alert System   | WebSocket + Notification API                  | 실시간 알림 시스템                  |

핵심: "시각화 디자인이 단순히 비주얼이 아니라, 실시간 데이터 파이프라인 구조를 고려한 설계"

## 핵심 지표 설명

> ⚠️ **참고 자료**: 이 프로젝트의 각 지표는 국제 표준 및 국내 공식 기관의 자료를 참고하였습니다.  
> 자세한 참고 자료는 [`참고.md`](./참고.md) 파일을 확인하시기 바랍니다.

### 기술·성능 지표

1. 저장 용량 (Nameplate Capacity): 전체 저장 가능한 에너지량
   - **참고**: IEC 62619:2022 (Secondary lithium batteries safety standards)
   - **관련 기관**: 한국에너지공단
2. 현재 전력 (Power in/out): 실시간 충전/방전 전력
   - **참고**: IEEE 1547:2018 (Distributed Energy Resources interconnection)
3. 충전 상태 (SOC): 현재 배터리 충전률
   - **참고**: IEC 62619:2022, "ESS 운영 가이드라인" - 한국에너지공단
4. 건강 상태 (SOH): 배터리 수명·성능 지표
   - **참고**: IEC 62620:2022, "배터리 성능 평가 기술" - 한국에너지공단
5. 효율 (RTE): 충전→저장→방전 손실률
   - **참고**: IEC 61427:2013, "ESS 효율 평가 지침" - 산업통상자원부
6. 사용 깊이 (DoD): 배터리 방전 깊이
   - **참고**: IEEE 1726:2011, "ESS 운영 가이드라인" - 한국에너지공단
7. 사이클 수: 충·방전 반복 횟수
   - **참고**: IEC 61959:2004 (배터리 기계적 테스트 표준)
8. 온도: 열관리 상태
   - **참고**: NFPA 855:2020 (ESS 설치 및 화재 안전 표준), "ESS 화재 안전 관리 가이드라인" - 소방청

### 에너지 관리 지표

1. 피크 셰이빙 (Peak Shaving): 전력 피크 시간대 부하 절감량
   - **참고**: "전력 수요관리 사업 운영 지침" - 한국에너지공단
   - **법령**: 전기사업법 제49조(수요관리)
2. 부하 평준화: 전력 사용 패턴의 균등화 정도
   - **참고**: "전력 수요 반응(DR) 프로그램 운영 규정" - 한국전력거래소, IEEE 1547:2018
3. 에너지 밸런싱: 수요-공급 균형 유지율
   - **참고**: "전력계통 운영 지침" - 한국전력거래소, "ESS 연계형 전력망 안정성 연구" - 한국전력연구원
4. 백업 전력 용량: 정전 시 공급 가능한 전력량
   - **참고**: IEEE 1699:2021, "전력 품질 및 비상 전원 관리" - 산업통상자원부

### 경제성 분석 지표

1. 전력 구매 비용 절감: 시간대별 요금 차이를 이용한 절감액
   - **참고**: "시간대별 전기요금제 운영 규정" - 한국전력공사
   - **법령**: 전기요금규정
2. 수요 반응 참여 수익: DR(Demand Response) 프로그램 수익
   - **참고**: "수요반응 실시계획" - 한국전력거래소, "수요반응 거래 규정" - 한국전력거래소
   - **관련 프로그램**: VPP(Virtual Power Plant)
3. 용량 요금 절감: 최대 수요 전력 감소로 인한 절감
   - **참고**: "전기요금 규정" - 산업통상자원부, "수요관리 전기요금제 운영 안내" - 한국에너지공단
4. 탄소 크레딧: CO₂ 감소로 인한 탄소 크레딧 수익
   - **참고**: "온실가스 감축실적 인정·관리 운영지침" - 환경부, K-ETS(한국형 온실가스 배출권 거래제)

### 운영 효율성 지표

1. 가동률 (Availability): 시스템 가동 가능 시간 비율
   - **참고**: IEC 61508 (Functional safety standards), "ESS 안정성 평가 지침" - 한국에너지공단
   - **목표 수준**: 95% 이상 권장
2. 신뢰성 지표: MTBF(평균 고장 간격), MTTR(평균 복구 시간)
   - **참고**: IEC 61710:2013, "ESS 신뢰성 평가 모델" - 한국전력연구원
3. 예측 정확도: 에너지 수요 예측의 정확도
   - **참고**: "스마트그리드 에너지 예측 알고리즘" - 한국전력연구원
   - **관련 기술**: AI 기반 에너지 수요 예측 시스템
4. 자동화 수준: 수동 개입 없이 자동 운영되는 비율
   - **참고**: "ESS 자동 운영 시스템 표준" - 산업통상자원부
   - **참고**: "스마트 EMS 운영 가이드라인" - 한국에너지공단

### 환경 및 지속가능성 지표

1. 재생에너지 연계율: 태양광/풍력 등과의 연계 비율
   - **참고**: "재생에너지 3020 이행계획" - 산업통상자원부
   - **관련 제도**: REC(신재생에너지공급인증서) 발급
   - **연구**: "ESS-재생에너지 연계 시스템 기술개발" - 산업통상자원부 R&D
2. 탄소 발자국: 전체 생명주기 동안의 CO₂ 배출량
   - **참고**: ISO 14040:2006 (Environmental management - Life cycle assessment)
   - **연구**: "배터리 생명주기 평가(LCA)" - 환경부
3. 재활용 가능성: 배터리 재활용 및 재사용 계획
   - **참고**: "전기차 배터리 재활용 가이드라인" - 환경부
   - **법령**: 자원순환기본법
   - **연구**: "폐배터리 재활용 기술개발" - 산업통상자원부 R&D
4. 친환경 인증: LEED, BREEAM 등 환경 인증 획득 현황
   - **참고**: LEED (Leadership in Energy and Environmental Design) - U.S. Green Building Council
   - **참고**: BREEAM (Building Research Establishment Environmental Assessment Method)
   - **국내**: "한국형 친환경 건축물 인증제도" - 국토교통부

### 운영·관리 지표

1. 활성 알림: 과충전, 과열, 네트워크 이상 등

2. 비용 절감: 에너지 저장을 통한 절감액

4. 배출량 감소: CO₂ 감소량

## 4. 경고 임계값 (Threshold)

### ESS 레벨 경고 조건
```
✓ 온도 > 30°C
✓ 가동률 < 95%
✓ 배터리 이상 감지 시
```

### 배터리 레벨 경고 조건
```
✓ 전압 < 3.5V
✓ 온도 > 35°C
✓ SOC < 20%
✓ SOH < 80%
```
